worldmonitor

打开本地服务 原仓库 项目文档

worldmonitor 是独立 Web 应用。当前已按原 README 拉取源码、安装依赖，并启动本地 Vite 服务；基础应用无环境变量即可运行，部分实时数据功能会在缺少 API Key 时降级或提示凭证缺失。

已部署：本地服务运行在 127.0.0.1:3100 本地路径：integrations/worldmonitor

计划保留能力

  • 500+ 策展新闻源
  • 3D globe 与 WebGL 地图层
  • 地缘、灾害、军事、金融信号交叉
  • 国家不稳定指数与金融雷达

运行方式

在 integrations/worldmonitor 中执行 npm run dev -- --host 127.0.0.1 --port 3100。当前首页的“进入项目”已直接指向本地服务。
